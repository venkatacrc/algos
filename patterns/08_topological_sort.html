Topological Sort Pattern

Topological Sort

üìñ Concept

Topological Sort orders nodes in a directed acyclic graph (DAG) such that for every edge (u, v), u appears before v in the ordering.

Key Idea: Process nodes with no incoming edges first, then remove their outgoing edges and repeat. Used for dependency resolution, course scheduling, build systems.

üéØ Techniques to Master

  • Kahn's Algorithm: Use in-degree counting and queue
  • DFS-based: Use DFS with post-order processing
  • In-degree Tracking: Count incoming edges for each node
  • Cycle Detection: Detect if graph has cycles (no valid topological order)
  • Dependency Resolution: Process dependencies before dependents
  • Queue Management: Process nodes with zero in-degree

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(V + E) V = vertices, E = edges. Visit each node and edge once
Space Complexity O(V) Space for in-degree array, queue, and result

üíª Boilerplate Code

Kahn's Algorithm (BFS-based):

from collections import deque

def topological_sort(num_courses, prerequisites):
    """
    Kahn's algorithm using in-degree counting
    """
    # Build graph and in-degree count
    graph = [[] for _ in range(num_courses)]
    in_degree = [0] * num_courses
    
    for course, prereq in prerequisites:
        graph[prereq].append(course)
        in_degree[course] += 1
    
    # Start with nodes having no incoming edges
    queue = deque([i for i in range(num_courses) if in_degree[i] == 0])
    result = []
    
    while queue:
        node = queue.popleft()
        result.append(node)
        
        # Remove edges and update in-degrees
        for neighbor in graph[node]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    # Check if all courses were processed (no cycle)
    return result if len(result) == num_courses else []

# TC: O(V + E)
# SC: O(V)

DFS-based Topological Sort:

def topological_sort_dfs(num_courses, prerequisites):
    """
    DFS-based topological sort
    """
    graph = [[] for _ in range(num_courses)]
    for course, prereq in prerequisites:
        graph[prereq].append(course)
    
    WHITE, GRAY, BLACK = 0, 1, 2
    color = [WHITE] * num_courses
    result = []
    
    def dfs(node):
        if color[node] == GRAY:  # Cycle detected
            return False
        if color[node] == BLACK:  # Already processed
            return True
        
        color[node] = GRAY
        for neighbor in graph[node]:
            if not dfs(neighbor):
                return False
        
        color[node] = BLACK
        result.append(node)
        return True
    
    for i in range(num_courses):
        if color[i] == WHITE:
            if not dfs(i):
                return []  # Cycle found
    
    return result[::-1]  # Reverse for correct order

# TC: O(V + E)
# SC: O(V)

Problem 1: Course Schedule

Description: Determine if you can finish all courses given prerequisites.

Example: numCourses = 2, prerequisites = [[1,0]] ‚Üí True

def can_finish(num_courses, prerequisites):
    graph = [[] for _ in range(num_courses)]
    in_degree = [0] * num_courses
    
    for course, prereq in prerequisites:
        graph[prereq].append(course)
        in_degree[course] += 1
    
    queue = deque([i for i in range(num_courses) if in_degree[i] == 0])
    count = 0
    
    while queue:
        node = queue.popleft()
        count += 1
        for neighbor in graph[node]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)
    
    return count == num_courses
