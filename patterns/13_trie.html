Trie Pattern

Trie (Prefix Tree)

üìñ Concept

A Trie (prefix tree) is a tree-like data structure for storing strings. Each node represents a character, and paths from root to nodes represent words.

Key Idea: Share common prefixes among words. Efficient for prefix matching, autocomplete, word search, and dictionary operations.

üéØ Techniques to Master

  • Node Structure: Each node has children (dict) and is_end flag
  • Insert Operation: Traverse/create path character by character
  • Search Operation: Check if word exists in trie
  • Prefix Search: Check if prefix exists
  • DFS Traversal: Find all words with given prefix
  • Space Optimization: Use array of size 26 for lowercase letters

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(m) m = length of word. Each operation traverses word length
Space Complexity O(ALPHABET_SIZE * N * M) N = number of words, M = average length. Can be optimized

üíª Boilerplate Code

Basic Trie Implementation:

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word):
        """
        Insert word into trie
        """
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end = True
    
    def search(self, word):
        """
        Check if word exists in trie
        """
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end
    
    def starts_with(self, prefix):
        """
        Check if prefix exists
        """
        node = self.root
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        return True

# TC: O(m) per operation
# SC: O(ALPHABET_SIZE * N * M)

class Trie:
    def __init__(self):
        self.root = {}
    
    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node:
                node[char] = {}
            node = node[char]
        node['#'] = True  # Mark end of word
    
    def search(self, word):
        node = self.root
        for char in word:
            if char not in node:
                return False
            node = node[char]
        return '#' in node
    
    def starts_with(self, prefix):
        node = self.root
        for char in prefix:
            if char not in node:
                return False
            node = node[char]
        return True

Problem 2: Word Search II

Description: Find all words from dictionary that exist in the board.

Example: board = [["o","a","a","n"],["e","t","a","e"],["i","h","k","r"],["i","f","l","v"]], words = ["oath","pea","eat","rain"] ‚Üí ["eat","oath"]

def find_words(board, words):
    trie = {}
    for word in words:
        node = trie
        for char in word:
            node = node.setdefault(char, {})
        node['#'] = word
    
    result = []
    
    def dfs(r, c, node):
        char = board[r][c]
        curr_node = node[char]
        
        word_match = curr_node.pop('#', False)
        if word_match:
            result.append(word_match)
        
        board[r][c] = '#'
        for dr, dc in [(0,1), (1,0), (0,-1), (-1,0)]:
            nr, nc = r + dr, c + dc
            if 0 <= nr < len(board) and 0 <= nc < len(board[0]) and board[nr][nc] in curr_node:
                dfs(nr, nc, curr_node)
        board[r][c] = char
        
        if not curr_node:
            node.pop(char)
    
    for i in range(len(board)):
        for j in range(len(board[0])):
            if board[i][j] in trie:
                dfs(i, j, trie)
    
    return result
