Union Find (Disjoint Set Union)

üìñ Concept

Union Find (Disjoint Set Union) is a data structure that tracks elements partitioned into disjoint sets. It supports efficient union and find operations.

Key Idea: Use path compression and union by rank to achieve near-constant time operations. Perfect for connectivity problems, cycle detection, and grouping elements.

üéØ Techniques to Master

  • Path Compression: Flatten tree during find operation for efficiency
  • Union by Rank: Always attach smaller tree to larger tree
  • Find Operation: Find root of element's set with path compression
  • Union Operation: Merge two sets by connecting their roots
  • Connected Check: Check if two elements are in same set
  • Count Components: Track number of disjoint sets

üíª Boilerplate Code

Union Find with Path Compression and Union by Rank:

class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.components = n
    
    def find(self, x):
        """
        Find root with path compression
        """
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        """
        Union by rank
        """
        root_x = self.find(x)
        root_y = self.find(y)
        
        if root_x == root_y:
            return False  # Already connected
        
        # Union by rank
        if self.rank[root_x] < self.rank[root_y]:
            self.parent[root_x] = root_y
        elif self.rank[root_x] > self.rank[root_y]:
            self.parent[root_y] = root_x
        else:
            self.parent[root_y] = root_x
            self.rank[root_x] += 1
        
        self.components -= 1
        return True
    
    def connected(self, x, y):
        """
        Check if two elements are connected
        """
        return self.find(x) == self.find(y)

# TC: O(Œ±(n)) per operation
# SC: O(n)

üìù Demonstrative Problems

Problem 1: Number of Connected Components

Description: Find number of connected components in an undirected graph.

Example: n = 5, edges = [[0,1],[1,2],[3,4]] ‚Üí 2 components

def count_components(n, edges):
    uf = UnionFind(n)
    
    for u, v in edges:
        uf.union(u, v)
    
    return uf.components

Problem 2: Redundant Connection

Description: Find redundant edge that forms a cycle in a tree.

Example: edges = [[1,2],[1,3],[2,3]] ‚Üí [2,3]

def find_redundant_connection(edges):
    n = len(edges)
    uf = UnionFind(n + 1)
    
    for u, v in edges:
        if not uf.union(u, v):
            return [u, v]  # Cycle detected
    
    return []

Problem 3: Number of Islands II

Description: Count islands after each land addition operation.

Example: m=3, n=3, positions=[[0,0],[0,1],[1,2],[2,1]] ‚Üí [1,1,2,3]

def num_islands_2(m, n, positions):
    uf = UnionFind(m * n)
    result = []
    grid = [[0] * n for _ in range(m)]
    directions = [(0,1), (1,0), (0,-1), (-1,0)]
    
    for r, c in positions:
        if grid[r][c] == 1:
            result.append(uf.components)
            continue
        
        grid[r][c] = 1
        uf.components += 1
        idx = r * n + c
        
        for dr, dc in directions:
            nr, nc = r + dr, c + dc
            if 0 <= nr < m and 0 <= nc < n and grid[nr][nc] == 1:
                neighbor_idx = nr * n + nc
                uf.union(idx, neighbor_idx)
        
        result.append(uf.components)
    
    return result
