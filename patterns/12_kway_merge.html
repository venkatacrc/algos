K-way Merge Pattern

K-way Merge

üìñ Concept

K-way Merge combines K sorted arrays/lists into one sorted array efficiently using a min-heap to track the smallest element from each array.

Key Idea: Use a min-heap to always get the smallest element across all K arrays. Pop smallest, add next element from same array, repeat until all arrays are exhausted.

üéØ Techniques to Master

  • Min-Heap Approach: Store (value, array_index, element_index) tuples
  • Pointer Tracking: Track current position in each array
  • Heap Initialization: Add first element from each array
  • Merge Process: Pop smallest, add next from same array
  • Array Exhaustion: Handle when an array is fully processed
  • Divide and Conquer: Merge pairs recursively for large K

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n log k) n = total elements, k = number of arrays. Each element processed once, heap operations O(log k)
Space Complexity O(k) Heap stores at most k elements (one from each array)

Merge K Sorted Lists:

import heapq

def merge_k_lists(lists):
    """
    Merge k sorted linked lists
    """
    heap = []
    
    # Initialize heap with first node from each list
    for i, lst in enumerate(lists):
        if lst:
            heapq.heappush(heap, (lst.val, i, lst))
    
    dummy = ListNode(0)
    current = dummy
    
    while heap:
        val, list_idx, node = heapq.heappop(heap)
        current.next = node
        current = current.next
        
        # Add next node from same list
        if node.next:
            heapq.heappush(heap, (node.next.val, list_idx, node.next))
    
    return dummy.next

# TC: O(n log k)
# SC: O(k)

Merge K Sorted Arrays:

def merge_k_arrays(arrays):
    """
    Merge k sorted arrays
    """
    heap = []
    result = []
    
    # Initialize: (value, array_index, element_index)
    for i, arr in enumerate(arrays):
        if arr:
            heapq.heappush(heap, (arr[0], i, 0))
    
    while heap:
        val, arr_idx, elem_idx = heapq.heappop(heap)
        result.append(val)
        
        # Add next element from same array
        if elem_idx + 1 < len(arrays[arr_idx]):
            next_val = arrays[arr_idx][elem_idx + 1]
            heapq.heappush(heap, (next_val, arr_idx, elem_idx + 1))
    
    return result

# TC: O(n log k)
# SC: O(k)

Problem 2: Kth Smallest in Sorted Matrix

Description: Find kth smallest element in a sorted matrix.

Example: matrix = [[1,5,9],[10,11,13],[12,13,15]], k = 8 ‚Üí 13

def kth_smallest(matrix, k):
    heap = []
    n = len(matrix)
    
    # Add first element of each row
    for i in range(n):
        heapq.heappush(heap, (matrix[i][0], i, 0))
    
    for _ in range(k - 1):
        val, row, col = heapq.heappop(heap)
        if col + 1 < n:
            heapq.heappush(heap, (matrix[row][col + 1], row, col + 1))
    
    return heap[0][0]
