Two Heaps Pattern

Two Heaps

üìñ Concept

The Two Heaps pattern uses a min-heap and max-heap to maintain elements in two halves, allowing efficient access to median, finding smallest/largest elements, or maintaining balanced partitions.

Key Idea: Use min-heap for larger half and max-heap for smaller half. Keep heaps balanced (size difference ‚â§ 1) for efficient median finding.

üéØ Techniques to Master

  • Min-Heap for Larger Half: Store larger elements in min-heap
  • Max-Heap for Smaller Half: Store smaller elements in max-heap (use negative values)
  • Balance Maintenance: Keep heap sizes balanced (difference ‚â§ 1)
  • Median Finding: Top of heaps give median when balanced
  • Re-balancing: Move elements between heaps to maintain balance
  • Python heapq: Use heapq (min-heap) with negatives for max-heap

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(log n) Insert and rebalance operations take O(log n)
Space Complexity O(n) Store all elements in two heaps

Find Median from Data Stream:

import heapq

class MedianFinder:
    def __init__(self):
        # Max heap (smaller half) - use negative values
        self.small = []
        # Min heap (larger half)
        self.large = []
    
    def add_num(self, num):
        # Add to appropriate heap
        if not self.small or num <= -self.small[0]:
            heapq.heappush(self.small, -num)
        else:
            heapq.heappush(self.large, num)
        
        # Balance heaps
        if len(self.small) > len(self.large) + 1:
            val = -heapq.heappop(self.small)
            heapq.heappush(self.large, val)
        elif len(self.large) > len(self.small) + 1:
            val = heapq.heappop(self.large)
            heapq.heappush(self.small, -val)
    
    def find_median(self):
        if len(self.small) > len(self.large):
            return -self.small[0]
        elif len(self.large) > len(self.small):
            return self.large[0]
        else:
            return (-self.small[0] + self.large[0]) / 2.0

# TC: O(log n) per operation
# SC: O(n)

Problem 1: Find Median from Data Stream

Description: Design a data structure that supports adding numbers and finding the median.

Example: addNum(1), addNum(2), findMedian() ‚Üí 1.5, addNum(3), findMedian() ‚Üí 2

class MedianFinder:
    def __init__(self):
        self.small = []  # Max heap (smaller half)
        self.large = []  # Min heap (larger half)
    
    def add_num(self, num):
        if not self.small or num <= -self.small[0]:
            heapq.heappush(self.small, -num)
        else:
            heapq.heappush(self.large, num)
        
        # Balance
        if len(self.small) > len(self.large) + 1:
            heapq.heappush(self.large, -heapq.heappop(self.small))
        elif len(self.large) > len(self.small) + 1:
            heapq.heappush(self.small, -heapq.heappop(self.large))
    
    def find_median(self):
        if len(self.small) == len(self.large):
            return (-self.small[0] + self.large[0]) / 2.0
        return -self.small[0] if len(self.small) > len(self.large) else self.large[0]

Problem 2: Sliding Window Median

Description: Find median for each window of size k in an array.

Example: nums = [1,3,-1,-3,5,3,6,7], k = 3 ‚Üí [1.0,-1.0,-1.0,3.0,5.0,6.0]

def median_sliding_window(nums, k):
    def get_median():
        if k % 2 == 1:
            return float(small[0][0] if len(small) > len(large) else -large[0][0])
        return (small[0][0] - large[0][0]) / 2.0
    
    small, large = [], []
    result = []
    
    # Initialize first window
    for i in range(k):
        heapq.heappush(small, (-nums[i], i))
    
    for i in range(k // 2):
        val, idx = heapq.heappop(small)
        heapq.heappush(large, (-val, idx))
    
    result.append(get_median())
    
    # Slide window
    for i in range(k, len(nums)):
        # Remove out-of-window elements
        while small and small[0][1] <= i - k:
            heapq.heappop(small)
        while large and large[0][1] <= i - k:
            heapq.heappop(large)
        
        # Add new element and balance
        if nums[i] <= -small[0][0] if small else True:
            heapq.heappush(small, (-nums[i], i))
        else:
            heapq.heappush(large, (nums[i], i))
        
        # Rebalance
        while len(small) > len(large) + 1:
            val, idx = heapq.heappop(small)
            heapq.heappush(large, (-val, idx))
        while len(large) > len(small) + 1:
            val, idx = heapq.heappop(large)
            heapq.heappush(small, (-val, idx))
        
        result.append(get_median())
    
    return result
