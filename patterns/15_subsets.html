Subsets Pattern

Subsets (Backtracking)

üìñ Concept

The Subsets pattern generates all possible subsets/combinations/permutations using backtracking. Each element can be included or excluded.

Key Idea: Use recursion to explore all possibilities. At each step, make a choice (include/exclude), recurse, then backtrack to try other choices.

üéØ Techniques to Master

  • Backtracking Template: Choose, explore, unchoose pattern
  • Include/Exclude: Two choices at each step
  • Duplicate Handling: Sort and skip duplicates
  • Path Tracking: Maintain current path/combination
  • Base Case: When to add current path to result
  • Pruning: Skip invalid paths early

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(2^n) For subsets: 2^n possible subsets (each element included/excluded)
Space Complexity O(n) Recursion stack depth and path storage

üíª Boilerplate Code

Generate All Subsets:

def subsets(nums):
    """
    Generate all subsets
    """
    result = []
    
    def backtrack(start, path):
        # Add current path to result
        result.append(path[:])
        
        # Try each remaining element
        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path)
            path.pop()  # Backtrack
    
    backtrack(0, [])
    return result

# TC: O(2^n)
# SC: O(n)

Subsets with Duplicates:

def subsets_with_dup(nums):
    """
    Generate subsets, skip duplicates
    """
    nums.sort()
    result = []
    
    def backtrack(start, path):
        result.append(path[:])
        
        for i in range(start, len(nums)):
            # Skip duplicates
            if i > start and nums[i] == nums[i - 1]:
                continue
            
            path.append(nums[i])
            backtrack(i + 1, path)
            path.pop()
    
    backtrack(0, [])
    return result

# TC: O(2^n)
# SC: O(n)

Combinations:

def combine(n, k):
    """
    Generate all combinations of k numbers from [1, n]
    """
    result = []
    
    def backtrack(start, path):
        if len(path) == k:
            result.append(path[:])
            return
        
        for i in range(start, n + 1):
            path.append(i)
            backtrack(i + 1, path)
            path.pop()
    
    backtrack(1, [])
    return result

# TC: O(C(n,k))
# SC: O(k)

Problem 3: Combination Sum

Description: Find all unique combinations that sum to target (can reuse elements).

Example: candidates = [2,3,6,7], target = 7 ‚Üí [[2,2,3],[7]]

def combination_sum(candidates, target):
    result = []
    
    def backtrack(start, path, remaining):
        if remaining == 0:
            result.append(path[:])
            return
        if remaining < 0:
            return
        
        for i in range(start, len(candidates)):
            path.append(candidates[i])
            backtrack(i, path, remaining - candidates[i])
            path.pop()
    
    backtrack(0, [], target)
    return result
