Bitwise XOR Pattern

Bitwise XOR

üìñ Concept

Bitwise XOR (^) has unique properties that make it useful for solving problems involving finding unique elements, swapping without temp, or detecting differences.

Key Properties: a ^ a = 0, a ^ 0 = a, a ^ b = b ^ a, (a ^ b) ^ c = a ^ (b ^ c). XOR of all elements cancels pairs.

üéØ Techniques to Master

  • Find Single Number: XOR all numbers, pairs cancel out
  • Swap Without Temp: a ^= b; b ^= a; a ^= b
  • Find Missing Number: XOR all numbers with range
  • Find Two Unique Numbers: XOR all, find rightmost set bit, partition
  • Complement Property: Use XOR with mask to flip bits
  • Parity Checking: XOR to check if count is odd/even

üíª Boilerplate Code

Find Single Number:

def single_number(nums):
    """
    Find single number in array where all others appear twice
    """
    result = 0
    for num in nums:
        result ^= num
    return result

# TC: O(n)
# SC: O(1)

Find Two Single Numbers:

def single_numbers(nums):
    """
    Find two numbers that appear once, others appear twice
    """
    # XOR all numbers
    xor_all = 0
    for num in nums:
        xor_all ^= num
    
    # Find rightmost set bit
    rightmost_bit = xor_all & -xor_all
    
    # Partition into two groups
    num1 = num2 = 0
    for num in nums:
        if num & rightmost_bit:
            num1 ^= num
        else:
            num2 ^= num
    
    return [num1, num2]

# TC: O(n)
# SC: O(1)

Swap Without Temporary Variable:

def swap(a, b):
    """
    Swap two numbers using XOR
    """
    a ^= b
    b ^= a
    a ^= b
    return a, b

# TC: O(1)
# SC: O(1)

Problem 2: Single Number III

Description: Find two numbers that appear once, others appear twice.

Example: nums = [1,2,1,3,2,5] ‚Üí [3,5]

def single_number_3(nums):
    xor_all = 0
    for num in nums:
        xor_all ^= num
    
    # Get rightmost set bit
    diff = xor_all & -xor_all
    
    num1 = num2 = 0
    for num in nums:
        if num & diff:
            num1 ^= num
        else:
            num2 ^= num
    
    return [num1, num2]

Problem 3: Missing Number

Description: Find missing number in array containing n distinct numbers from [0, n].

Example: nums = [3,0,1] ‚Üí 2

def missing_number(nums):
    result = len(nums)
    for i, num in enumerate(nums):
        result ^= i ^ num
    return result
