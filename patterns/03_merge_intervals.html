Merge Intervals Pattern

Merge Intervals

üìñ Concept

The Merge Intervals pattern is used to solve problems involving overlapping intervals. The key is to sort intervals by their start time and then merge or find relationships between them.

Key Idea: Sort intervals by start time, then iterate through them, merging overlapping intervals or finding gaps/conflicts.

üéØ Techniques to Master

  • Sort First: Always sort intervals by start time (or end time for specific problems)
  • Overlap Detection: Two intervals overlap if start[i] <= end[j] and start[j] <= end[i]
  • Merge Strategy: Compare current interval with last merged interval
  • Gap Finding: Identify gaps between non-overlapping intervals
  • Conflict Detection: Find intervals that conflict with each other
  • Insert Intervals: Insert new interval and merge if necessary

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n log n) Sorting takes O(n log n), merging takes O(n)
Space Complexity O(n) Space needed for storing merged intervals

üíª Boilerplate Code

Basic Merge Intervals:

def merge_intervals(intervals):
    """
    Merge overlapping intervals
    """
    if not intervals:
        return []
    
    # Sort by start time
    intervals.sort(key=lambda x: x[0])
    merged = [intervals[0]]
    
    for current in intervals[1:]:
        last = merged[-1]
        # If current overlaps with last, merge them
        if current[0] <= last[1]:
            merged[-1] = [last[0], max(last[1], current[1])]
        else:
            merged.append(current)
    
    return merged

# TC: O(n log n)
# SC: O(n)

Insert Interval:

def insert_interval(intervals, new_interval):
    """
    Insert new interval and merge if necessary
    """
    result = []
    i = 0
    
    # Add all intervals before new_interval
    while i < len(intervals) and intervals[i][1] < new_interval[0]:
        result.append(intervals[i])
        i += 1
    
    # Merge overlapping intervals
    while i < len(intervals) and intervals[i][0] <= new_interval[1]:
        new_interval[0] = min(new_interval[0], intervals[i][0])
        new_interval[1] = max(new_interval[1], intervals[i][1])
        i += 1
    
    result.append(new_interval)
    
    # Add remaining intervals
    result.extend(intervals[i:])
    return result

# TC: O(n)
# SC: O(n)

Problem 2: Insert Interval

Description: Insert a new interval into a sorted list of non-overlapping intervals and merge if necessary.

def insert(intervals, new_interval):
    result = []
    i = 0
    
    while i < len(intervals) and intervals[i][1] < new_interval[0]:
        result.append(intervals[i])
        i += 1

        new_interval[0] = min(new_interval[0], intervals[i][0])

Problem 3: Meeting Rooms II

Description: Find the minimum number of conference rooms required.

Example: intervals = [[0,30],[5,10],[15,20]] ‚Üí Output: 2

def min_meeting_rooms(intervals):
    if not intervals:
        return 0
    
    start_times = sorted([i[0] for i in intervals])
    end_times = sorted([i[1] for i in intervals])
    
    start_ptr = end_ptr = 0
    rooms = 0
    max_rooms = 0
    
    while start_ptr < len(intervals):
        if start_times[start_ptr] < end_times[end_ptr]:
            rooms += 1
            start_ptr += 1
        else:
            rooms -= 1
            end_ptr += 1
        max_rooms = max(max_rooms, rooms)
    
    return max_rooms
