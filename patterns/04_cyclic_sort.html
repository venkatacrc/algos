Cyclic Sort Pattern

Cyclic Sort

üìñ Concept

Cyclic Sort is an in-place sorting algorithm that works well when dealing with arrays containing numbers in a given range. It places each number at its correct index.

Key Idea: For an array of n elements with values in range [0, n-1] or [1, n], each value should be at index equal to its value. We swap elements to their correct positions.

üéØ Techniques to Master

  • Range-Based Sorting: Works best when numbers are in a known range
  • Index Mapping: Value at index i should be i (or i+1 for 1-based)
  • Swap Strategy: Swap current element to its correct position
  • Missing Number Detection: After sorting, find index where value != index
  • Duplicate Detection: Identify duplicates during sorting process
  • Boundary Checks: Ensure indices are within valid range before swapping

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n) Each element is swapped at most once to its correct position
Space Complexity O(1) Sorting is done in-place, only using constant extra space

üíª Boilerplate Code

Basic Cyclic Sort (0 to n-1):

def cyclic_sort(arr):
    """
    Sort array where numbers are in range [0, n-1]
    """
    i = 0
    while i < len(arr):
        # If number is at correct position, move to next
        if arr[i] == i:
            i += 1
        else:
            # Swap to correct position
            arr[i], arr[arr[i]] = arr[arr[i]], arr[i]
    
    return arr

# TC: O(n)
# SC: O(1)

Cyclic Sort (1 to n):

def cyclic_sort_1_to_n(arr):
    """
    Sort array where numbers are in range [1, n]
    """
    i = 0
    while i < len(arr):
        correct_pos = arr[i] - 1
        if arr[i] != arr[correct_pos]:
            arr[i], arr[correct_pos] = arr[correct_pos], arr[i]
        else:
            i += 1
    
    return arr

# TC: O(n)
# SC: O(1)

Find Missing Number:

def find_missing_number(arr):
    """
    Find missing number in array containing n distinct numbers from range [0, n]
    """
    i = 0
    while i < len(arr):
        if arr[i] < len(arr) and arr[i] != i:
            arr[i], arr[arr[i]] = arr[arr[i]], arr[i]
        else:
            i += 1
    
    # Find first index where value != index
    for i in range(len(arr)):
        if arr[i] != i:
            return i
    
    return len(arr)  # n is missing

# TC: O(n)
# SC: O(1)

def missing_number(nums):

    while i < len(nums):
        if nums[i] < len(nums) and nums[i] != i:
            nums[i], nums[nums[i]] = nums[nums[i]], nums[i]

            i += 1

Problem 2: Find All Duplicates

Description: Find all duplicates in an array where numbers are in range [1, n].

Example: nums = [4,3,2,7,8,2,3,1] ‚Üí Output: [2,3]

def find_duplicates(nums):
    i = 0
    while i < len(nums):
        correct_pos = nums[i] - 1
        if nums[i] != nums[correct_pos]:
            nums[i], nums[correct_pos] = nums[correct_pos], nums[i]
        else:
            i += 1
    
    duplicates = []
    for i in range(len(nums)):
        if nums[i] != i + 1:
            duplicates.append(nums[i])
    
    return duplicates
