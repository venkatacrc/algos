Tree DFS Pattern

Tree DFS (Depth-First Search)

üìñ Concept

DFS traverses a tree by going as deep as possible before backtracking. It uses recursion or a stack to explore paths.

Key Idea: Explore one path completely before backtracking. Three main orders: Pre-order (root-left-right), In-order (left-root-right), Post-order (left-right-root).

üéØ Techniques to Master

  • Recursive DFS: Use function calls to naturally traverse depth-first
  • Iterative DFS: Use stack to simulate recursion
  • Pre-order: Process root before children (root-left-right)
  • In-order: Process left, then root, then right (for BST gives sorted order)
  • Post-order: Process children before root (left-right-root)
  • Path Tracking: Maintain current path while traversing

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n) Visit each node exactly once
Space Complexity O(h) h is height of tree (recursion stack or explicit stack)

üíª Boilerplate Code

Pre-order Traversal (Recursive):

def preorder_traversal(root):
    """
    Pre-order: root -> left -> right
    """
    result = []
    
    def dfs(node):
        if not node:
            return
        result.append(node.val)
        dfs(node.left)
        dfs(node.right)
    
    dfs(root)
    return result

# TC: O(n)
# SC: O(h)

In-order Traversal (Recursive):

def inorder_traversal(root):
    """
    In-order: left -> root -> right
    """
    result = []
    
    def dfs(node):
        if not node:
            return
        dfs(node.left)
        result.append(node.val)
        dfs(node.right)
    
    dfs(root)
    return result

# TC: O(n)
# SC: O(h)

Post-order Traversal (Recursive):

def postorder_traversal(root):
    """
    Post-order: left -> right -> root
    """
    result = []
    
    def dfs(node):
        if not node:
            return
        dfs(node.left)
        dfs(node.right)
        result.append(node.val)
    
    dfs(root)
    return result

# TC: O(n)
# SC: O(h)

Iterative DFS (Pre-order):

def preorder_iterative(root):
    """
    Pre-order using stack
    """
    if not root:
        return []
    
    result = []
    stack = [root]
    
    while stack:
        node = stack.pop()
        result.append(node.val)
        
        # Push right first, then left (so left is processed first)
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    
    return result

# TC: O(n)
# SC: O(h)

üìù Demonstrative Problems

Problem 1: Maximum Depth of Binary Tree

Description: Find the maximum depth of a binary tree.

Example: [3,9,20,null,null,15,7] ‚Üí 3

def max_depth(root):
    if not root:
        return 0
    
    left_depth = max_depth(root.left)
    right_depth = max_depth(root.right)
    
    return 1 + max(left_depth, right_depth)

Problem 2: Path Sum

Description: Check if there exists a root-to-leaf path with given sum.

Example: root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22 ‚Üí True

def has_path_sum(root, target_sum):
    if not root:
        return False
    
    if not root.left and not root.right:
        return root.val == target_sum
    
    remaining = target_sum - root.val
    return (has_path_sum(root.left, remaining) or 
            has_path_sum(root.right, remaining))

Problem 3: Binary Tree Maximum Path Sum

Description: Find the maximum path sum (path can start and end at any node).

Example: [-10,9,20,null,null,15,7] ‚Üí 42

def max_path_sum(root):
    max_sum = float('-inf')
    
    def dfs(node):
        nonlocal max_sum
        if not node:
            return 0
        
        left_sum = max(0, dfs(node.left))
        right_sum = max(0, dfs(node.right))
        
        # Path through current node
        current_path = node.val + left_sum + right_sum
        max_sum = max(max_sum, current_path)
        
        # Return max path from this node (can only use one branch)
        return node.val + max(left_sum, right_sum)
    
    dfs(root)
    return max_sum
