Two Pointers Pattern

Two Pointers

üìñ Concept

The Two Pointers pattern uses two pointers that traverse the data structure in a coordinated way. This pattern is highly efficient for solving problems involving arrays, strings, or linked lists.

Key Idea: Use two pointers moving at different speeds or from different directions to efficiently solve problems without nested loops.

üéØ Techniques to Master

  • Opposite Ends: Start pointers at opposite ends and move them toward each other
  • Fast & Slow: Use two pointers moving at different speeds (Floyd's cycle detection)
  • Same Direction: Both pointers move in the same direction at different speeds
  • Sliding Window: Two pointers define a window that slides through the array
  • Partitioning: Use pointers to partition array into sections
  • Cycle Detection: Fast pointer catches slow pointer if cycle exists

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n) Each element is visited at most once by each pointer
Space Complexity O(1) Only using a constant amount of extra space for pointers

üíª Boilerplate Code

Opposite Ends (Two Sum in Sorted Array):

def two_sum_sorted(arr, target):
    """
    Find two numbers that add up to target in sorted array
    """
    left, right = 0, len(arr) - 1
    
    while left < right:
        current_sum = arr[left] + arr[right]
        if current_sum == target:
            return [left, right]
        elif current_sum < target:
            left += 1
        else:
            right -= 1
    
    return []

# TC: O(n)
# SC: O(1)

Fast & Slow (Cycle Detection):

def has_cycle(head):
    """
    Detect cycle in linked list using Floyd's algorithm
    """
    if not head or not head.next:
        return False
    
    slow = fast = head
    
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    
    return False

# TC: O(n)
# SC: O(1)

Same Direction (Remove Duplicates):

def remove_duplicates(arr):
    """
    Remove duplicates in-place from sorted array
    """
    if not arr:
        return 0
    
    write_ptr = 1
    
    for read_ptr in range(1, len(arr)):
        if arr[read_ptr] != arr[read_ptr - 1]:
            arr[write_ptr] = arr[read_ptr]
            write_ptr += 1
    
    return write_ptr

# TC: O(n)
# SC: O(1)

Problem 1: Two Sum II - Input Array Is Sorted

Description: Given a sorted array, find two numbers that add up to a specific target.

Example: numbers = [2, 7, 11, 15], target = 9 ‚Üí Output: [1, 2]

def two_sum(numbers, target):
    left, right = 0, len(numbers) - 1
    
    while left < right:
        current_sum = numbers[left] + numbers[right]
        if current_sum == target:
            return [left + 1, right + 1]  # 1-indexed
        elif current_sum < target:
            left += 1
        else:
            right -= 1
    
    return []

Problem 3: Container With Most Water

Description: Find two lines that together with the x-axis forms a container that holds the most water.

Example: height = [1,8,6,2,5,4,8,3,7] ‚Üí Output: 49

def max_area(height):
    left, right = 0, len(height) - 1
    max_water = 0
    
    while left < right:
        width = right - left
        current_area = min(height[left], height[right]) * width
        max_water = max(max_water, current_area)
        
        # Move pointer with smaller height
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    
    return max_water
