Tree BFS Pattern

Tree BFS (Breadth-First Search)

üìñ Concept

BFS traverses a tree level by level, visiting all nodes at the current depth before moving to the next level. It uses a queue to process nodes.

Key Idea: Process nodes level by level using a queue. Start with root, add children to queue, process each level completely before moving to next.

üéØ Techniques to Master

  • Queue-Based: Use queue (deque) to maintain nodes at current level
  • Level Tracking: Process all nodes at current level before moving to next
  • Level Order Traversal: Return nodes grouped by level
  • Zigzag Traversal: Alternate direction at each level
  • Right Side View: Track last node at each level
  • Level Size: Track number of nodes at each level for processing

‚è±Ô∏è Time & Space Complexity

Complexity Type Value Explanation
Time Complexity O(n) Visit each node exactly once
Space Complexity O(w) w is maximum width of tree (queue size at widest level)

üíª Boilerplate Code

Basic BFS:

from collections import deque

def bfs(root):
    """
    Basic BFS traversal
    """
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        node = queue.popleft()
        result.append(node.val)
        
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
    
    return result

# TC: O(n)
# SC: O(w)

Level Order Traversal:

def level_order(root):
    """
    Return nodes grouped by level
    """
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        level = []
        
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        result.append(level)
    
    return result

# TC: O(n)
# SC: O(w)

Zigzag Level Order:

def zigzag_level_order(root):
    """
    Traverse tree in zigzag pattern
    """
    if not root:
        return []
    
    result = []
    queue = deque([root])
    left_to_right = True
    
    while queue:
        level_size = len(queue)
        level = deque()
        
        for _ in range(level_size):
            node = queue.popleft()
            
            if left_to_right:
                level.append(node.val)
            else:
                level.appendleft(node.val)
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        result.append(list(level))
        left_to_right = not left_to_right
    
    return result

# TC: O(n)
# SC: O(w)

Problem 2: Binary Tree Right Side View

Description: Return the values of nodes visible from the right side.

Example: [1,2,3,null,5,null,4] ‚Üí [1,3,4]

def right_side_view(root):
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        
        for i in range(level_size):
            node = queue.popleft()
            
            # Last node at this level
            if i == level_size - 1:
                result.append(node.val)
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
    
    return result

Problem 3: Average of Levels in Binary Tree

Description: Return the average value of nodes at each level.

Example: [3,9,20,null,null,15,7] ‚Üí [3.0, 14.5, 11.0]

def average_of_levels(root):
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        level_sum = 0
        
        for _ in range(level_size):
            node = queue.popleft()
            level_sum += node.val
            
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        result.append(level_sum / level_size)
    
    return result
